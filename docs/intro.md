# Tutorial Intro

Let's run through **Chiron in less than 5 minutes**.

## Getting Started

Get started by downloading Chiron. You can do this through `Cargo`, the package manager for Rust.

First, visit [rustup](https://rustup.rs/) to install Rust.

Second, run the following command:

```bash
cargo install chiron
```

## Using Chiron

From here, you can use any of the programs from Chiron simply by typing their name in the command window.

For example:

```bash
c3dio
```
